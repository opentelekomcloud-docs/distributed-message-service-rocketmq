Querying Messages
=================

Function
--------

This API is used to query messages.

URI
---

Example Requests
----------------

This API is used to query messages.

.. code-block:: text

   GET https://{endpoint}/v2/{project_id}/rocketmq/instances/{instance_id}/messages?topic={topic}&&msg_id=C0A807C9000027740000000000000030
